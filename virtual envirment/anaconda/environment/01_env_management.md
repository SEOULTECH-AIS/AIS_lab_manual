# Conda를 이용한 가상환경 생성과 관리

해당 페이지에서는 Conda 명령어를 통해 다양한 가상환경을 실제로 구성하고, 이를 관리하는 기본적인 방법을 정리하고 있음.

## 가상환경 생성

``` bash
conda create [-n {env_name}] []
```

| 구분자 | 기능 |
| :---: | :---: |
| -n | 가상환경 이름|
| -p | 지정한 경로로 가상환경이 위치함<br> (가상환경 기본 경로 ```$codna_path/env/$env_name```)|
| -c | 채널 지정 |
| --file | |
| --clone | |

### 예시

#### 단순 환경 생성

```bash
conda create -n ${env_name} 
```

### 별도의 참조를 사용한 환경 생성

```bash
conda create
```
